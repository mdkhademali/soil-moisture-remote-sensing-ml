
## Methodology

1. Collect satellite imagery (Sentinel-2 / Landsat).
2. Extract indices:
   - NDVI
   - NDWI
   - Land Surface Temperature proxy
3. Combine with ground soil moisture measurements.
4. Train machine learning models.
5. Evaluate performance using RMSE and R².

Potential data sources:
- NASA SMAP
- Google Earth Engine exports
- FAO soil datasets
