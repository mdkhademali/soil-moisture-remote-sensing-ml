
## Soil Moisture Estimation using Remote Sensing & Machine Learning

A research-style project demonstrating how satellite-derived indices and environmental variables
can be used to estimate soil moisture using machine learning.

## Main Features
- Remote sensing index calculation (NDVI, NDWI, LST proxy)
- MATLAB data preprocessing pipeline
- Machine learning models:
  - Random Forest
  - Support Vector Regression
  - Neural Network
- Model evaluation and visualization
- Optional Python preprocessing for satellite rasters
